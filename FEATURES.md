# MemoryVerse - Feature Roadmap & Ideas

**Last Updated:** November 17, 2025

## üéØ **Planned Features (Prioritized)**

### High Priority - V1.1

#### 1. Prayer Input Safety & Moderation
- Input validation before AI processing
- Content filtering for inappropriate language
- Character limits (200-1000 characters)
- Rate limiting per user per day
- Abuse logging and monitoring
- Fallback messages for rejected inputs

#### 2. Streak Freeze Feature
- Allow premium users to freeze streak when unable to practice
- Basic: 1 freeze per month
- Standard: 3 freezes per month
- Premium: Unlimited freezes
- UI to activate/view freeze status

#### 3. Advanced Verse Analytics
- Track accuracy over time per verse
- Show learning curve graphs
- Identify difficult verses
- Personalized practice recommendations

#### 4. All Bible Translations
- Currently: ESV only
- Add: NIV, KJV, NKJV, NLT, NASB
- Allow users to switch translations
- Store favorite translation in profile

---

### Medium Priority - V1.2

#### 5. Social Features
- Share progress with friends
- Compete on leaderboards
- Group challenges
- Encourage friends feature

#### 6. Study Plans
- Pre-built verse collections (e.g., "Verses on Faith", "Psalms for Comfort")
- Topic-based memorization tracks
- Custom study plans users can create
- Progress tracking per plan

#### 7. Offline Mode
- Download verses for offline practice
- Sync progress when back online
- Works during flights, poor connection

#### 8. Audio Features
- Text-to-speech for verse reading
- Record yourself reciting verses
- Compare your recording to correct text
- Audio-only practice mode

---

### Low Priority - V2.0

#### 9. Community Features
- Share custom verse lists
- Comment on verses
- Prayer requests from community
- Devotional content integration

#### 10. Smart Notifications
- Remind based on user's best practice time
- "You're about to lose your streak!" alerts
- Personalized verse suggestions
- Daily verse push notifications

#### 11. Verse of the Day Customization
- Choose verse categories
- Difficulty preference
- Testament preference (OT/NT)
- Topic-based rotation

#### 12. Enhanced Gamification
- Badges/achievements system
- Daily challenges
- Weekly competitions
- Milestone celebrations
- Avatar customization
